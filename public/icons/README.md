# PWA Icons

For the PWA to work optimally on all devices, you should generate PNG icons in the following sizes:

- 72x72
- 96x96
- 128x128
- 144x144
- 152x152
- 192x192
- 384x384
- 512x512

## Recommended Icon Design

The icon should:
- Be square (will be masked to a circle on most devices)
- Have a solid background color matching your brand
- Include a simple logo or letter
- Be at least 512x512 for best quality

## Quick Generation Tools

You can use online tools like:
- https://realfavicongenerator.net/
- https://www.favicon-generator.org/

## Current Setup

Currently using SVG icon (icon.svg) which works for modern browsers.
For iOS compatibility, PNG icons are recommended.
